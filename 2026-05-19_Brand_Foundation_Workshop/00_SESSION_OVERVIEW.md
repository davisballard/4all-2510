## WHY V2 (different from Jerk Gel Feb 2026)

You loved Jerk Gel. The agency was in infancy then — flat 8-person room, single-day push, voice + aesthetic baked together, no formal handoff to next-phase deliverables.

The agency has matured a lot since. V2 leans into that:

| Then (Jerk Gel) | Now (4 All Snacks V2) |
|---|---|
| Flat 8-person room throughout | **Per-block roundtable rooms** with named chairs per phase |
| Workshop opened cold with audit | **Nigel bring-in protocol** opens — formal echo of Ishan's brief + room assembly |
| Audit framing: "your docs are thin" | **Audit framing: "your deck is RICH — here's what crystallized vs what's still open"** |
| No formal handoff | **Wrap previews the 8-doc Campaign Book stack** |
| One day, one push | **Day 1 (~10-10.5h with lunch) locks foundation; Day 2 (~4h) refines + opens pre-launch** |
| Voice + Aesthetic combined | **Voice + Aesthetic split** so Fink/Massimo own visual world separately from McCabe's voice work |
| No pre-launch / audience-build block | **Day 2 PM block: Pre-Launch Marketing Strategy** — directly serves Ishan's "if we have time in the afternoon" ask |

V2 is **harder on the agency, lighter on the founders** — most of the diagnostic work is done in pre-read. The in-room time is for *decisions and generation,* not for relitigating the category.

## HOW THIS WORKS

This is a real workshop. The agency leads. You're founders AND collaborators.

- **Nigel** opens each block by assembling the right room.
- **The chair of the block** drives the work; other voices contribute on their specialty.
- **Silent observers stay quiet** unless they have something load-bearing to add — then they interrupt themselves with "(pulled in)".
- **Davis is head of agency** — addressed as such, not "the client." Ishan + Elliot are the founders.
- **The team will debate each other** — that's the point. When experts disagree, the insight lives in the seam.
- **Write everything down.** Verbal isn't strategy.
- **If something clicks, STOP and capture it before moving on.**

## HOW TO RUN

1. **Start by firing `01_NIGEL_OPENING.md`** — Nigel walks in, echoes Ishan's brief, assembles the room. Don't skip beats; don't collapse beats; the opening *is* the workshop's spine.
2. **Move through blocks in order** — each block builds on the last. The chair of each block names the room before opening it.
3. **Take breaks when the room earns them** — not on a clock.
4. **If a block runs long because you're onto something, LET IT RUN.** The schedule is scaffolding, not law.
5. **If a block resolves quickly, move on** — don't pad.
6. **Fire each block file when ready for that section.**
7. **Write everything down.** Verbal agreement isn't strategy.
8. **Captures go in `SESSION_NOTES.md`** — parking lot, divergent ideas, raw quotes. Don't pollute the block files with stuff that's off-topic for that block.

**Remember:** The agency shares opinions FIRST, then asks for yours. This is a back-and-forth, not a questionnaire. Nigel's job in beat 1 of each block is to make sure that's clear — *the room speaks before the founders speak.*
